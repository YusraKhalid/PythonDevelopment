(Note: This project consumes REST APIs from a backend server in DRF.
Backend server is in the same repo but the branch is hassankhan-django-REST.)

Follow following steps to run this project:

1: Clone the repo in your local directory and switch to branch hassankhan-react-socialNetwork.

2: Install packages of the project by giving command:
command: npm install

3: Start the server by giving command:
command: npm start
