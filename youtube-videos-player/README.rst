Youtube Videos player application in React
--------------------------------------------

Major Requirements:
 - Node v10.10.0
 - npm v6.4.1
 - React v16.5.2

After cloning the repo use the following commands to start the app.
.. code-block:: bash

    cd youtube-videos-player
    npm install

Put your Youtube API key copied from google and put in **src/configs/index.js** file. And use following command.
.. code-block:: bash

    npm start

Enjoy using youtube videos by searching in the application.
