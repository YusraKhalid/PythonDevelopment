After redirecting to the directory of the program, type in
    python3 main.py -s <your query to search> -n <number f tweets to search>

By default,
    number of tweets ot search = 5
    Query to search = Fifa 2018

Once the tweets are searched, the category of each tweet will be listed below it. Tweets are classified into following listed categories
decreasing in priority:
1- Link
2- Hash tag
3- Mention
4- Text
